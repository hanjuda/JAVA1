서술형) "인터프리터 방식 설명 / 이 방식에 해당하는 대표적인 언어"
	- 소스코드를 한줄씩 읽고 (명령을) 하나씩 해석하여 처리" 
	  별도의 사전변환 과정 없이 소스코드 상태로 하나씩 읽어서 기계어로 번역하여 처리
	  명령문을 하나씩 번역하고 실행
	/ 파이썬

서술형) "객체"와 "클래스"를 설명
	- 객체 : 클래스의 인스턴스
		 클래스를 기반으로 힙 영역에 생성된 데이터의 실체
	- 클래스 : 객체의 틀(설계도)
		   객체의 타입을 정의

서술형) "싱글턴(Singleton) 패턴"을 설명
	- 오직 하나의 인스턴스만을 가지도록 하는 패턴
